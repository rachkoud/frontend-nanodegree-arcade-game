Class: Sprite

Sprite

new Sprite(Url, Position, Size, speed, frames, dir, once)

This sprite animation class has been take from here : Website : http://jlongster.com/Making-Sprite-based-Games-with-Canvas GitHub : https://github.com/jlongster/canvas-game-bootstrap
Parameters:
Name Type Description
Url String Url of the sprite
Position Array.<Number> of the entity on the sprite [x, y]
Size Array.<Number> of the entity on the sprite [width, height]
speed Number Speed in frames/sec for animating
frames Array.<Number> An array of frame indexes for animating: [0, 1, 2, 1]
dir String Which direction to move in the sprite map when animating: 'horizontal' (default) or 'vertical'
once Boolean True to only run the animation once, defaults to false

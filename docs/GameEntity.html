Class: GameEntity

GameEntity

new GameEntity(pos, spritePos, spriteSize, collisionPosOffset, collisionSize, spriteSpeed, spriteFrames, spriteDir, spriteRenderOnce)

A GameEntity is base class for render a game entity on the board, example : new GameEntity([0, 0, [0, 1269], [101, 79], [16,17], [67,76], 16, [0, 1, 2, 3, 4], null, true)
Parameters:
Name Type Description
pos Array.<number> Position of the entity on the canvas [x,y]
spritePos Array.<number> Position of the entity on the sprite [x,y]
spriteSize Array.<number> Site of the entity on the sprite [width, height]
collisionPosOffset Array.<number> Collision rectancle offset position [offsetX,offsetY]
collisionSize Array.<number> Collision rectangle size [width, height]
spriteSpeed number Speed in frames/sec for animating
spriteFrames Array.<number> An array of frame indexes for animating: [0, 1, 2, 1]
spriteDir string Which direction to move in the sprite map when animating: 'horizontal' (default) or 'vertical'
spriteRenderOnce boolean True to only run the animation once, defaults to false
